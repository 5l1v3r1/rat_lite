<ul>
<li>
Cross-platform:Windows desktop, windows phone, Linux, Modern Web Browsers, Andriod, Mac desktop, iphone
</li>
<li>
Performance 
</li>
<li>
Encryption (openssl)
</li>
<li>
Latest standards: c++ 17 
</li>
</ul>
